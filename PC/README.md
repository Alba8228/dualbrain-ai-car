# 智能车跟随 - PC端AI视觉模块

## 项目简介

本模块是智能小车项目的上位机AI视觉部分，基于YOLOv8n目标检测和SORT多目标跟踪算法，实现人体识别与跟踪功能，并通过Socket发送控制指令给OpenWRT，最终控制STM32小车跟随目标移动。

## 技术架构

```

│                                                 └─────┬─────┘  │
```

## 核心功能

1. **人体检测** - 使用YOLOv8n轻量级模型快速检测画面中的人体目标
2. **多目标跟踪** - SORT算法结合卡尔曼滤波和匈牙利算法，实现稳定的目标跟踪
3. **智能决策** - 根据检测框位置和面积计算方向偏移与距离，生成控制指令
4. **Socket通信** - 将控制指令发送给OpenWRT，实现远程控制

## 算法说明

### SORT跟踪算法

- **卡尔曼滤波** - 预测目标状态，平滑轨迹
- **匈牙利算法** - 二分图匹配，最小化IOU匹配代价
- **IOU计算** - 交并比，衡量检测框与预测框相似度

### 决策策略

```
画面中心 = width / 2

IF bbox_center_x < 画面中心 - 死区:
    → 左转
ELSE IF bbox_center_x > 画面中心 + 死区:
    → 右转
ELSE:
    → 直行

IF bbox_area < 最小面积阈值:
    → 前进
ELSE IF bbox_area > 最大面积阈值:
    → 后退
ELSE:
    → 停止
```

## 通信协议

控制字节格式：
- 帧头：`AA 55`
- 指令字节：小车方向(00-04) + 舵机方向(05) + 速度(0A-0C)
- 校验和：所有字节之和取低8位

| 指令 | 字节 | 说明 |
|------|------|------|
| 前进 | 01 | 小车前进 |
| 后退 | 02 | 小车后退 |
| 左转 | 03 | 小车左转 |
| 右转 | 04 | 小车右转 |
| 停止 | 00 | 小车停止 |
