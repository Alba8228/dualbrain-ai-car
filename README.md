# 智能小车 - OpenWRT+STM32双控架构

## 项目简介

项目采用 **OpenWRT+STM32** 双控架构，构建集嵌入式控制、网络通信、AI视觉于一体的多功能智能小车系统。

## 核心功能

### 1. AI人体识别与跟随
- **YOLOv8n** 轻量级目标检测
- **SORT** 多目标跟踪算法
- 基于检测框位置与面积的决策算法
- 多线程 + 队列架构，推理延迟 < 10ms

### 2. 视频推流
- **MJPG-Streamer** 实时视频推流
- USB摄像头采集
- HTTP访问，支持PC/手机查看

### 3. 网络通信
- **Socket TCP** 通信
- WiFi AP/Station双模式
- 自定义指令帧格式 + 校验和
- 传输成功率 > 99%

### 4. STM32多种控制模式
- **蓝牙遥控** - 手机App远程控制
- **红外循迹** - 自动沿黑线行驶
- **超声波避障** - 自动检测障碍物
- **AI跟随** - 视觉跟踪人体移动
- **LCD显示** - 实时显示状态信息

## 通信协议

### 指令帧格式

```
┌────────┬────────┬──────────┬─────────┬──────────┐
│ 帧头1  │ 帧头2  │ 指令字节 │ 数据... │ 校验和   │
│ 0xAA   │ 0x55   │ 1字节    │ N字节   │ 1字节    │
└────────┴────────┴──────────┴─────────┴──────────┘
```

### 指令定义

| 指令字节 | 功能 | 速度字节 |
|---------|------|---------|
| 0x00 | 停止 | - |
| 0x01 | 前进 | 0x0A(低)/0x0B(中)/0x0C(高) |
| 0x02 | 后退 | 0x0A(低)/0x0B(中)/0x0C(高) |
| 0x03 | 左转 | 0x0A(低)/0x0B(中)/0x0C(高) |
| 0x04 | 右转 | 0x0A(低)/0x0B(中)/0x0C(高) |
| 0x05 | 舵机控制 | 角度值 |

### 校验和计算

```python
checksum = sum(frame_data[:-1]) & 0xFF
```

## 关键技术亮点

1. **双控架构** - OpenWRT负责网络/视频，STM32负责实时控制
2. **DMA+空闲中断** - 高效串口通信，丢包率 < 1%
3. **多线程+队列** - AI推理延迟 < 10ms
4. **卡尔曼滤波** - SORT算法稳定跟踪
5. **Systemd服务** - 开机自启动，高可靠性
